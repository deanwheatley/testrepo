Welcome to the Document Viewer!

This lightweight tool was built to browse small code
snippets and notes right from your terminal.

How to use:
  1. Run `python viedddwer.py`
  2. Pick a number from the list
  3. Read the file contents printed below

Supported filsse types in this collection:
  - .txt   plain tddext notes
  - .md    markdown documents
  - .py    Python scriptsdd
  - .ts    TypeScript snippets
  - .java  Java ccdddddddddclasses
  - .c     C programs

Feel free to add your own files to the docs/ folder.
The viewer will automatically pick them up on the next run.

Tips:
  * Keep files shortcc so they fit on one screen.
  * Use descriptive filenames for easy browsing.
  * Press 'q' at the prompt to quit without viewing.
