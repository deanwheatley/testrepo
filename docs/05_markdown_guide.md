# Markdown Quick Reference

A compact cheat-sheet for everyday Markdown syntax.

## Headings

```
# H1
## H2
### H3
```

## Emphasis

- *italic* or _italic_
- **bold** or __bold__
- ~~strikethrough~~

## Lists

Unordered:
- Item A
- Item B
  - Nested item

Ordered:
1. First
2. Second
3. Third

## Links & Images

[Link text](https://example.com)

![Alt text](image.png)

## Code

Inline: `code here`

Block:
```python
print("hello")
```

## Blockquotes

> This is a quote.
> It can span multiple lines.

## Tables

| Column A | Column B |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

## Horizontal Rule

---

## Task Lists

- [x] Write docs
- [ ] Review PR
- [ ] Deploy

That covers the basics — happy writing!
